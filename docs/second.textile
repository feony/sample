h4. @bool swap(int takeMask, int pushMask)@

Снять одни флаги и установить другие указанные в масках; аналогично вызову двух методов подряд: @#take(int)@ и @#push(int)@.

|takeMask|integer|Биты, которые нужно забрать из корзинки.|
|pushMask|integer|Биты, которые нужно положить в корзинку.|

Пример: @[0101'0101] => swap(1100'0000, 0011'0000) => 0011'0101 => return true@

h4. @bool turn(int mask, int model)@

Установить флаги по маске в соответствии с моделью. Меняет в корзинке все флаги, которые отмечены параметром @mask@ на те значения, которые указаны в параметре @model@.

|mask|integer|Биты, которые нужно изменить в корзинке.|
|model|integer|Новые значения бит отмеченных маской.|

Пример: @[0101'0101] => turn(1111'0000, 1100'0000) => [1100'0101] => return true@
